@Component - помечаем ей класс,если хотим что бы spring содал нам бин этого класса.
Можно указать в параметрах id этого бина.Если не указывать id,то ему присваиваться id - имя класса с маленькой буквы.

@Autowired - Используется для внедрения зависимостей в spring.Спринг ищет подходящий бин,и сам внедряет зависимость.
Может размещаться над сеттером,конструктором или нам самим полем.В проекте выбрать только один способ создания,и использовать его.

@Qualifier - Указываем id того бина,который хотим внедрить.Может размещаться над сеттером,конструктором или нам самим полем.
Для конструктора размещается непосредственно перед параметром.
public Cat( @Qualifier("cat")Animal animal, @Qualifier("name") String name){}

@Value - Позволяет задавать значение полям класса.

@Scope - Задает scope нашим полям.(Singleton or prototype).

@PostConstruct - Вызывает указанный init method.Методы не должны принимать на вход какие либо значения(т.е параметры должн ыбыть пустые).Модификатор доступа
может быть любым.Имя метода может быть любым.Тип возвращаемого значения может быть любым.

@PreDestroy - Вызывает указанный Destroy метод.Для prototype scope дестрой метод не вызывается.Методы не должны принимать на вход какие либо значения(т.е параметры должн ыбыть пустые).
Модификатор доступа может быть любым.Имя метода может быть любым.Тип возвращаемого значения может быть любым.

@Configuration - Помечает Java class,который мы хотим использовать для конфигурации Spring приложения.

@Bean - В configuration классе помечается бин как в xml файле.
