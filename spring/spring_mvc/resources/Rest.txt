REST - это паттерн проектирования web - приложений.Он описывает,как посредством протокола HTTP должен взаимодействовать клиент
с сервером.Все взаимодействия с сервером сводятся к 4 операциям: Получение данных,добавление новых данных,имзенение существующих
данных,удаление данных и др. Для каждой из 4 операций используется свой HTTP метод: GET,POST,PATCH,DELETE.

CRUD - четыре базовые функции,используемые для работы с БД.
