Цель: Определить вид создаваемых объектов с помощью экземпляра - прототипа и создавать новые объекты,копирую этот прототип.

Используется для создания копий заданного объекта.

Пример использования:
- Классы ,экземпляры которых необходимо создать определяются во время выполнения программы.
- Для избежания построения иерархии классов,фабрик или параллельных иерархий классов.
- Экземпляры класса могут находится в одном из немногих возможный состояний.
